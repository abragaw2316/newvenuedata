## Working Filters (derived in Phase 1 — the scoring criteria for Phase 5)

The highest-probability solo profile is a niche that is:
1. **B2B-entity data, not consumer/people data** (sidesteps FCRA + state data-broker laws).
2. **Public-but-painful source** (fragmented county/municipal/agency records) → moat via assembly labor.
3. **Signals a near-term commercial transaction** → high WTP, recurring need.
4. **Self-serve-friendly** (dev/SMB buyer; metered API + programmatic-SEO funnel) → no sales org.
5. **High update frequency** → freshness moat + recurring-revenue justification.
6. **AI-automatable acquisition** (scrape + PDF→JSON + entity-resolution).
7. **Expandable** into intelligence / lead-gen / predictive layers.

**Hard avoid (Phase 1 legal findings):** anything used for credit/employment/tenant/insurance eligibility (FCRA); DMV/driver records (DPPA); health (HIPAA); financial account data (GLBA); sensitive consumer-location data (active FTC enforcement).
